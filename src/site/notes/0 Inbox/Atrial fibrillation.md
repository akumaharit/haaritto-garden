## Classification
- EHRA Scale #mnemonics 
	- 1 None -> ไม่มีอาการ
	- 2A Mild -> มีอาการ
	- 2B Moderate -> มีอาการแต่รำคาญเฉย ๆ
	- 3 Severe -> กระทบ daily life
	- 4 Disabling

## Risk Factor of Stroke in AF Patient
- Hypertension
- Ageing
- Structural Heart Disease
- Diabetes mellitus
- Vascular Disease
- CHF/LV Dysfunction
- Sex Category (female)
## Acute Treatment and control
- [[0 Inbox/Antiarrhythmic\|Antiarrhythmic]]
- Rate control
	- Stabilize hemodynamic, if unstable -> prefer ablation therapy
	- stable -> 
		- use beta-blocker 1st line
		- alternative: NonDHP CCB, Digoxin and Amiodarone IV short course for last option
		- asthma, copd -> NonDHP CCB
		- พวก beta-blocker -> Class II (beta-blockers)
		- พวก NonDHP CCB -> Class IV (calcium)
		- HFrEF -> beta-blockers
- Rhythm control
	- cardioversion
		- Direct current cardioversion
		- via medication (Ic and III)
			- Class Ic (sodium channel, slow on-off) -> Flecainide, Propafenone
			- Class III (potassium channel) -> Amiodarone (also class I, II, IV)
				- amiodarone -> CYP3A4 inhibitors, p-gp inhibitors
			- โดย HF -> Amiodarone
			- ถ้าไม่มีโรคร่วม or HTN -> ใช้ 1c

## AF แต่เสือกเป็น NSTEMI + PCI
- Triple therapy ได้ (ASA + P2Y12 inh) **+ NOAC or VKA**
## Management
### A - Anticoagulant (avoid stroke)
![Pasted image 20240311231450.png|650](/img/user/3%20Resources/Attachment/Pasted%20image%2020240311231450.png)
- CHA2DS2-VASc #mnemonics 
	- C =  CHF
	- H = Hypertension or on Hypertensive Medication
	- A = Age > 75 (2)
	- D = DM
	- S = Stroke (2)
	- V = Vascular Disease
	- A = Age 65-74
	- Sc = Sex Category Female 
- HAS-BLED #mnemonics 
	- Hypertension (uncontrolled)
	- Abnormal renal or liver (2)
	- Stroke
	- Bleeding Tendency or predisposition
	- Labile INR (unstable)
	- Age > 65
	- Drugs

![Pasted image 20230716153756.png|625](/img/user/3%20Resources/Attachment/Pasted%20image%2020230716153756.png)
- ใช้ HAS-BLED ในการประเมินความเสี่ยง Bleeding แต่ไม่ว่าจะเสี่ยงแค่ไหน ถ้าไม่ได้มี Absolute Contraindication ก็ควรใช้ OAC **เพราะ Benefit มันสูงกว่ามาก ๆ**
	- ถ้า HAS-BLED >= 3 = High Risk of Bleeding ให้พิจารณาใช้ Low Dose OACs แทน
- Absolute Contraindication เช่น Active Bleeding, Recent Intracranial Hemorrhage, Severe Thrombocytopenia

### What to use for stroke prevention therapies?
- Valvular AF -> Warfarin
- Non-valvular -> CHA2DS2VASc (>= 2 in male or 3 in female)
	- CrCl < 15 ml/min -> Warfarin or Apixaban (apixaban พบว่าใช้ใน intermittent dialysis ได้)
	- ถ้ามีประวัติ Stroke -> NOACs เลย

### [[0 Inbox/Warfarin\|Warfarin]] Management
แนะนำเริ่มที่ 3-5 mg/day
before surgey: หยุด 4-5 วัน / หาก INR > 1.5 ให้ VitK before surgery
หลัง surgery: bridging anticoagulation
